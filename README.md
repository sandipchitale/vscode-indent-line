# indent-line README

An extension to indent lines to correct level.

## Features

It supports the following commands:

- Indent line               (windows: `cmd+alt+[` , mac: `cmd+option+[` )

- Indent line and next line (windows: `cmd+alt+]` , mac: `cmd+option+]` )
